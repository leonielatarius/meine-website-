Auszeichnungssprachen

In diesem Kurs beschäftige ich mich mit verschiedenen Auszeichnungssprachen und deren Einsatz in Webseiten, Datenstrukturen und Grafiken.

HTML

Mit HTML habe ich die Grundstruktur meiner Webseite erstellt. Dazu gehören mehrere Unterseiten, Navigation, Überschriften, Absätze, Listen, Tabellen und Formulare.

XML

Mit XML habe ich gelernt, strukturierte Daten zu beschreiben. Ein Beispiel dafür ist eine Filmsammlung, die mit eigenen Elementen und Attributen modelliert werden kann.

Filmsammlung als XML ansehen

  • Baumstruktur eines XML-Dokuments
  • wohlgeformte XML-Dokumente
  • Elemente und Attribute
  • XML-Schema zur Validierung

SVG

Mit SVG habe ich Vektorgrafiken direkt im HTML-Code erstellt, zum Beispiel ein Logo und weitere grafische Elemente.

CSS

Mit CSS wird die Webseite gestaltet. Dazu gehören Farben, Abstände, Schriftgrößen, Layouts und responsive Darstellung.

Programmieren 2

In diesem Kurs arbeite ich mit Java. Ein Beispiel ist ein Burger-Projekt, das später in diesem Portfolio genauer dokumentiert werden kann.

Interaction Design

In diesem Kurs beschäftige ich mich mit Konzepten, Personas, Prototypen und der Gestaltung interaktiver Produkte. Ein Beispiel ist eine Webseite für ein imaginäres Startup.
